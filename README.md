\# Anatolia Industry — Corporate Landing Page



A bilingual (TR/EN) corporate website built for \*\*Anatolia Industry\*\*, an industrial engineering company specializing in pipe manufacturing, pipeline welding, refinery projects, and energy infrastructure.



🔗 \*\*Live site:\*\* \[anatoliaindustry.com](https://anatoliaindustry.com)



\## Overview



This project was developed as a freelance engagement — a full corporate presence covering services, featured projects, company statistics, and a contact section, with real-time language switching between Turkish and English.



\## Features



\- 🌐 \*\*Bilingual support (TR/EN)\*\* — custom language context with instant switching, no page reload

\- 🎨 \*\*Modern, responsive UI\*\* — built with Tailwind CSS and shadcn/ui (Radix primitives)

\- 🌓 \*\*Dark / light theme\*\* support

\- 📩 \*\*Contact form\*\* with client-side validation (react-hook-form + zod)

\- 📊 \*\*Analytics\*\* integration via Vercel Analytics

\- ⚡ \*\*Optimized performance\*\* — Next.js App Router, image optimization, SEO-friendly metadata



\## Tech Stack



| Layer | Technology |

|---|---|

| Framework | Next.js 16 (App Router), React 18, TypeScript |

| Styling | Tailwind CSS, shadcn/ui, Radix UI |

| Forms \& Validation | react-hook-form, zod |

| Icons | lucide-react |

| Deployment / Analytics | Vercel |



\## Project Structure



app/            → App Router pages, global layout \& styles

components/     → Header, Hero, Services, Projects, Statistics, About, Contact, Footer

components/ui/  → shadcn/ui component library

lib/            → Language context (i18n) and utilities

hooks/          → Custom React hooks

public/         → Static assets and images



\## Getting Started



npm install

npm run dev



Open http://localhost:3000 to view it locally.
